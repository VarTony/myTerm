## Description
Remake of the old project that was written in Express.js.

## Installation (Clone by HTTPS):

```bash
https://github.com/VarTony/myTerm.git
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
